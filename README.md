# global-mortality-regression
Global Mortality Regression Analysis

This project analyzes global adult mortality rates using statistical modeling.

Dataset
World Health Organization global health data (2000–2015)

Methods
- Multiple linear regression
- Residual diagnostics
- Box-Cox transformation
- Outlier analysis

Key findings
Development status, HIV prevalence, alcohol consumption, and education
were significant predictors of adult mortality.

Tools
R
Statistical modeling
Data visualization
